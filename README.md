##RichEditor

RichEditor is a rich text editor work on Android.
Until now,it support these styles:

- Bold
- Italic
- Underline
- Strikethrough 
- Quote 
- Bullet 
- FontSize
- Align Left
- Align Center
- Align Right 
- Todos
- Insert Image

All Styles can export as HTML , as well  it also can parse HTML to native weight  then edit or display.
